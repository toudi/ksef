Ta strona przeprowadzi Cię przez jedną z podstawowych funkcjonalności programu czyli import plików z fakturami

{{< callout type="warning" >}}
Jest bardzo istotnym abyś był świadomy, że program **nie** przeprowadza walidacji merytorycznej Twoich faktur. Innymi słowy, stosuje zasadę "garbage in, garbage out". Oczywiście pokażę Ci jak przeprowadzić walidację semantyczną faktury za pomocą programu `xmllint` tym niemniej ważne jest abyś był świadomy, że program nie ostrzeże Cię o błędach merytorycznych w fakturze
{{< /callout >}}
