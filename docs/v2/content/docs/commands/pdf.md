# `pdf`

Użyj komendy aby wygenerować PDF dla UPO lub faktury. Domyślnie PDF zostanie zapisany w tym samym katalogu w którym znajduje się źródłowy dokument. Możesz jednak nadpisać plik wyjścia

Obsługiwane flagi

| flaga             | znaczenie                                |
| ----------------- | ---------------------------------------- |
| `--output` / `-o` | Nadpisz lokalizację docelowego pliku PDF |
